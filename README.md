# README

* Ruby version
    - 3.2.1

* Run
 - bundle install
 - rails db:create
 - rails db:migrate
 - rails db:seed

* App:
 - rails s

* Specs:
 - bundle exec rspec


 Steps:
 1) Visit http://localhost:3000/admin and log in as the default user:

 User: admin@example.com
 Password: password

 2) Add new account in accounts tab
 3) Add new transaction for account in Financial Transactions tab
